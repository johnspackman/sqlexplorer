Update Release Info:

- edit sqlexplorer/change_log.txt
- set new version in plugin.xml/feature.xml of all changed project
- edit sqlexplorer_feature/feature.xml witn new version and change log informations and synchronize versions

Build update site:

- go to sqlexplorer_updatesite
- delete all files from features and plugins directories
- open site.xml
- set new version for sqlexplorer feature
- run 'Build All' from site.xml 
- run build.xml p2 to update artifacts.xml and content.xml (path to eclipse 3.5 executable must be configured)

Build standalone distribution:

- go to sqlexplorer_rcp
- run build.xml zip.plugin to build plugin.zip and xml.zip

Deploy files
- sftp zip files from dist to uploads
- login to shell
- go to uploads
- run build_sr.sh [version number]
- check dist
- run dist.sh to copy to project web
- login to sf admin page
- add release info to files

Finalize
- commit all changes
- tag ALL projects with the new version
